Windows users

If you‚Äôre getting weird installation errors, you might need to try a different method and install from source.

  1. Load these packages in your R session. If all is working as it should, you may get a message similar to Linking to GEOS 3.11.0, GDAL 3.5.3, PROJ 9.1.0 when loading the sf package (version numbers may differ).
